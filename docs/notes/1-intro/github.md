# Git, GitHub, Codespaces

- Git: version control software program. Tracks changes to files in a repository.
- GitHub: web service that lets you store your Git repositories. Provides additional features on top of that.
- Codespaces: a workspace and online code editor hosted by GitHub that let's you edit your repositories using your web browser.

In this lecture we will cover how to accept homework assignments, work on them, commit and push your changes, and verify automated test results.
