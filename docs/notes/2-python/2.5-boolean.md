## Boolean Logic

### Summary

Elementary algebra

- Operators: `+`, `-`, `*`, `/`
- Inputs: numbers
- Output: number
- e.g. `1 + 2 = 3`

Boolean algebra

- Operators: `AND`, `OR`, `XOR`, etc.
- Inputs: `True` or `False`
- Output: `True` of `False`
- e.g. `True AND False = False`

Operators

- `AND`: Both inputs must be `True`
- `OR`: At least one input must be `True`
- `XOR`: Exactly one input must be `True`
- `NOT`: Flip the input (only has one input)

## Boolean in Python

### Data Types

Python supports a Boolean data type. The supported values are `True` or `False`.

```python
isOpen = True
print(isOpen)
print(type(isOpen))

isOpen = False
print(isOpen)
```

You can convert other data types into Booleans. This means converting a value into either `True` or `False`.

In Python:

- 0 maps to `False`
- Empty string (`""`) maps to `False`
- `None` maps to `False`
- Most everything else maps to `True`

```python
print(0, bool(0))
print(0.0, bool(0.0))
print(1, bool(1))
print(2, bool(2))
print("", bool(""))
print("a", bool("a"))
print("0", bool("0")) # Careful!
print("False", bool("False")) # Careful!
```

### Expressions

An expression that evaluates to a number is an arithmetic expression:

```python
#    3 * 2 + 1    This is an arithmetic expression.
#                 It evaluates to a number.
print(3 * 2 + 1)
```

An expression that evaluates down to a True or False is a Boolean expression.

```python
#    True or False   This is a Boolean expression.
#                    It evaluates to a Boolean.
print(True or False)
```

### Logical Operators

You can use `and`, `or`, and `not` keywords in Python.

```python
print(True and True)
print(True and False)
print(True or True)
print(True or False)
print(not True)
print(not False)
```

There is no `xor`, but we'll see how you can achieve that later.

You can use these with an combination of Boolean values.

```python
a = True
b = False
print(a and b)
print(a or b)
print(a and True)
print(a and bool(0))
```
